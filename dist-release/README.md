# CopyGum v1.0.0

A modern clipboard manager for macOS with a beautiful overlay interface.

## Installation

1. **Download** `CopyGum_1.0.0_aarch64.dmg`
2. **Open** the DMG file
3. **Drag** CopyGum to your Applications folder
4. **Launch** CopyGum from Applications
5. **Grant Accessibility Permission** when prompted (required for global shortcuts)

## First Launch Setup

On first launch, macOS will ask for **Accessibility permissions**:

1. Go to **System Settings** > **Privacy & Security** > **Accessibility**
2. Click the **+** button
3. Navigate to **Applications** > **CopyGum**
4. Enable the toggle for CopyGum

This permission is required for the global keyboard shortcut to work.

## Usage

| Action | Shortcut |
|--------|----------|
| Toggle CopyGum Panel | `Cmd + Shift + V` |
| Copy item to clipboard | Click on any card |
| Pin item | Click the pin icon |
| Delete item | Click the trash icon |

## Features

- **Smart Content Detection** - Automatically categorizes URLs, emails, code, colors, and more
- **System App Icons** - Shows the actual app icon where content was copied from
- **Categories & Tags** - Organize your clipboard history
- **Image Support** - Saves images with thumbnails and dominant color extraction
- **Search** - Quickly find any clipboard item
- **Privacy Mode** - Blur sensitive content
- **Global Shortcut** - Access from any app with Cmd+Shift+V

## System Requirements

- macOS 10.15 (Catalina) or later
- Apple Silicon (M1/M2/M3) or Intel Mac

## Troubleshooting

### Shortcut not working?
1. Check that CopyGum has Accessibility permission
2. Try quitting and relaunching CopyGum
3. Check if another app is using the same shortcut

### Panel not appearing?
1. Make sure CopyGum is running (check menu bar)
2. Verify Accessibility permission is granted
3. Try pressing `Cmd + Shift + V` again
